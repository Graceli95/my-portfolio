# Event Images

This directory contains images for events, conferences, hackathons, and community involvement activities.

## Organization

Store event images with descriptive names:
- `year-up-ambassador-1.jpg`
- `year-up-ambassador-2.jpg`
- `commit-your-code-1.jpg`
- `commit-your-code-2.jpg`
- `hackathon-name-1.jpg`
- etc.

## Guidelines

- **Format**: JPG or PNG recommended
- **Size**: Optimize images to 1200px max width for web performance
- **Quantity**: 3-4 images per event for best results
- **Content**: Show yourself at events, team photos, presentations, or venue shots

## Image Naming Convention

Use the pattern: `[event-slug]-[number].[ext]`

Examples:
- `year-up-ambassador-1.jpg`
- `year-up-ambassador-2.jpg`
- `commit-your-code-conference-1.jpg`
- `hackathon-2024-1.jpg`
